Postupi AI

Postupi AI — сервис для абитуриентов российских вузов, который анализирует публичные конкурсные списки и помогает оценить реальный шанс поступления.

Обычная позиция в списке не всегда показывает реальную ситуацию: абитуриенты выше могут пройти на направления с более высоким приоритетом, изменить согласие или освободить место. Postupi AI отслеживает изменения в конкурсных списках, сопоставляет публичные данные между конкурсными группами и показывает понятный прогноз по каждому направлению.
Возможности

    Отслеживание конкурсных списков вузов

    Сохранение истории изменений списков

    Анализ текущей позиции и числа доступных мест

    Учёт согласий на зачисление и приоритетов

    Оценка эффективной позиции абитуриента

    Вероятностный прогноз поступления с объяснением факторов

    Уведомления в Telegram о значимых изменениях

    Поддержка нескольких вузов и направлений

    Админ-инструменты для контроля парсеров и источников

Принципы приватности

Postupi AI работает только с публично доступными конкурсными списками.

    Исходные уникальные идентификаторы абитуриентов не сохраняются в базе данных

    Для технического сопоставления используется HMAC-хеширование ID

    Сервис не показывает чужие ID, баллы, заявления, направления или предполагаемые предпочтения

    Нет поиска по чужому идентификатору

    Пользователь может удалить свои данные

    Прогноз не является гарантией зачисления
